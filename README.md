# MCUME-C64-8-BIT-parallel-lcd-320x240
MCUME emulator modified to work with a 3.2 inch 8 bit parallel 321x240 LCD 

original code:  github.com/Jean-MarcHarvengt/MCUME/

test video : https://youtu.be/2G1GnmDUAys

In this project, the SD reader integrated into the LCD display is used. Inside the SD insert only .PRG files organized into multiple folders. the initial menu displays the contents of the SD.

functions:
Virtual keyboard.
Menu setting: joystic, audio, palette, user function button:
Ability to delete files from the SD.
